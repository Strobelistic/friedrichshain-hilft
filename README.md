# Friedrichshain hilft Hugo theme

## Development

### Setup

You will need [hugo](http://gohugo.io) and [node.js](http://nodejs.com) to start.
Then...

```sh
npm i
grunt
```

### Deployment

Need the setup above and then:

```sh
npm run-script deploy
```
